# Diff

Write a Rust program that checks two text files for equivalence. If the two files contain the same text, write `equal` on a single line, otherwise write `different`.

## Example 1

file1:
    
    foo bar
    baz
    
file2:

    foo bar
    baz
    
Output:

    equal
    

# Example 2

file1:
    
    foo bar
    baz
    
file2:

    Hello, world!
    
Output:

    different
